Working with cargo

Now, let's work with Cargo - Rust's package manager

How to get Cargo

Cargo is automatically installed with rustup.


                cargo -h // get help for cargo

How to create Rust project?

Rust projects are created via Cargo

cargo new my-project

They also can be initialised in existing directory

cargo init

How to build and run Rust projects?

Rust project as you probably think are also built with Cargo

cargo build

Than, binary is in target folder

.targer/bin/my-project

But usually that process can be simplified with cargo run which build binary and runs it

cargo run

cargo also supports a wide range of features, like managing dependencies, built-in tests and documentation autogeneration

What is Rustbook?

Rustbook is a part of the official documentation for Rust. It's introduction to start writing Rust code. Rustbook
